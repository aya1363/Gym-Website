# FitCore Gym Website

## Overview

FitCore Gym is a modern, responsive landing page designed to showcase a premium fitness facility. This project demonstrates front-end development skills using HTML5, CSS3, and best practices for web design. The website highlights the gym's features, benefits, member testimonials, and provides a membership sign-up form.

## Features

- **Hero Section**: Eye-catching introduction with call-to-action buttons
- **Why Choose Us**: Detailed benefits including premium equipment, expert trainers, and flexible hours
- **Testimonials**: Real member stories with statistics
- **Membership Form**: Interactive sign-up form with validation
- **Responsive Design**: Optimized for various screen sizes
- **Modern Typography**: Uses Google Fonts (Roboto) for a clean, professional look

## Technologies Used

- HTML5
- CSS3
- Google Fonts API

## How to View

1. Clone or download the project files
2. Open `index.html` in any modern web browser
3. The website is fully static and requires no server setup
